PAMS Desktop App Start Instructions
===================================

1) Install dependencies
   pip install -r requirements.txt

2) Ensure MySQL is running and credentials are correct
   Default used by app:
   - Host: 127.0.0.1
   - Port: 3306
   - User: root
   - Password: Saboora.1
   - DB: pams

3) Initialize database + mock data
   python3 scripts/bootstrap_db.py

4) Run the application
   python3 main.py
