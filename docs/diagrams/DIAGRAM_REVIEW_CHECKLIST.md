# Final Diagram Review Checklist

## Use Case

- [ ] All required actors included
- [ ] All required components represented
- [ ] Functional coverage matches brief
- [ ] Diagram is self-explanatory

## Class Diagram

- [ ] Attributes + methods for key classes
- [ ] Relationship multiplicities shown
- [ ] Naming conventions consistent
- [ ] Visibility and data types shown

## Sequence Diagrams

- [ ] 3+ diagrams included
- [ ] Each maps to one use case
- [ ] Lifelines/messages/activations clean

## Submission Formatting

- [ ] All diagrams in ONE PDF
- [ ] PDF opens correctly on another machine
- [ ] Font size readable at 100% zoom
