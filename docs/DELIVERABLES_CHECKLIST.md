# Portfolio Submission Checklist (Task 1)

## Element 1: Design (35)

- [ ] One PDF containing all diagrams
- [ ] Use Case Diagram (complete actors + use cases)
- [ ] Class Diagram (attributes, methods, relationships, visibility)
- [ ] At least 3 Sequence Diagrams (one use case each)
- [ ] Diagrams created by your group (not AI-generated design)

## Element 2: Development + Methodology (30)

- [ ] Source code included
- [ ] GUI desktop app (Tkinter) working
- [ ] MySQL schema + data provided
- [ ] Security features shown
- [ ] Agile methodology evidence included
- [ ] Contribution evidence included
- [ ] Run/start instructions included
- [ ] Additional package list included

## Element 3: Testing (10)

- [ ] Manual test cases table
- [ ] Automated tests output included
- [ ] Evidence screenshots included
- [ ] Invalid data/boundary tests included

## Viva + Demo (Task 2)

- [ ] All 5 group members attend
- [ ] System demo script rehearsed
- [ ] Roles and contributions can be explained
- [ ] Design decisions can be explained without AI dependency
